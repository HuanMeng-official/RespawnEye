## 🤔什么是RespawnEye？
**RespawnEye** 是一个基于Minecraft 1.20.4开发的 **Fabric** Mod，同时它在MIT LICENSE下获得许可。

## 🎮如何安装或使用RespawnEye？
安装 **Fabric Loader** 和 **Fabric API** 并将本模组放入 **mods** 文件夹。

## 🔨构建RespawnEye：
克隆此模组的源代码并将其导入到您的集成开发环境中，然后运行 **Build** 并等待构建完成。
