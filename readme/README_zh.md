## 介绍：
*“想象一下，在面对巨大的神话生物时，你会有多么兴奋？准备好开始一段奇妙的冒险了吗？”*
RespawnEye 是一个专注于探索的 Fabric 模组。其主要特点如下：  
 - 酣畅淋漓的 Boss 战
 - 独特的神话生物
 - 丰厚的探索奖励
 - 以及更多！

## FAQ：
1. 它为什么崩溃了？  
回答：不要安装OptiFine，它会导致各种错误。
2. 可以将它与 OptiFine 一起使用吗？  
回答：可以，但我并不推荐。
